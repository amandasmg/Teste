TEMA BLUE_UFBA

Tema do Moodle baseado no tema custom_corners e customizado especificamente para o Moodle da Universidade Federal da Bahia.

Autores: Equipe do Projeto EAD - CPD - Moodle UFBA.

OBS.: Para utilizar os ícones do tema, copie todas as imagens da pasta imagens_novas para a pasta pix/i/ dentro do diretório principal de instalação do Moodle.
Se quiser reverter as imagens para as originais do Moodle, fazer o mesmo procedimento acima, mas utilizando as imagens da pasta imagens_originais.
